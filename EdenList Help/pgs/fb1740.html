Turning fonts on

Fonts that have been disabled or turned off don't appear in the Font panel or in the list of available fonts in an application.

If you want disabled fonts to appear in these places, you can turn them on in Font Book. Disabled fonts have "Off" next to their names in the Font column.

  1. Double-click Font Book in the Finder, or choose Manage Fonts from the Action pop-up menu in the lower-left corner of the Font panel.
  2. Select the fonts you want to turn on in the Font column.

    Select an entire collection in the Collection column, or select individual fonts in the Font column.

  3. Choose Edit > Enable Fonts or Edit > Enable Font Collections.
