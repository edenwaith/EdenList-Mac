A dot appears in my fonts list

A dot next to the name of a font family in Font Book indicates that you have duplicate copies of that font family installed. Font Book can easily resolve this problem by disabling any extra versions of the font family.

To resolve duplicate fonts:

  1. In Font Book, select the version of the font family that you want to use.
  2. Choose Edit > Resolve Duplicates.

The selected version of the font is turned on while all other versions are turned off.
