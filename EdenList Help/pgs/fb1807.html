Creating a new font library or collection from text in a document

You can use Font Book to quickly create a font collection or a font library that contains the fonts used in a selection of text.

Not all applications support this feature. To see if your application supports it, check the Services menu for the application to see if it includes the Font Book services.

  1. Open the document.
  2. Select all or some of the text in the document. Make sure you select text formatted in each font you want to add to your collection.
  3. In the application menu, choose Services > Font Book > Create Collection From Text or Create Font Library From Text.
  4. Open Font Book and locate the new collection or library in the Collection column.
  5. Type a new name for the collection or library.
