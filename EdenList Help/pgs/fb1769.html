Making fonts available only to you

If there are fonts installed on the computer that you don't want others to use, you can use Font Book to restrict access to these fonts.

To restrict access to a font:

  1. Double-click Font Book in the Finder, or choose Manage Fonts from the Action pop-up menu in the lower-left corner of the Font panel.
  2. Select Computer in the Collection column, and then select the font in the Font column.
  3. Drag the font to the User collection in the Collection column.
  4. Select the font in the Computer collection and click the Disable button or choose Edit > Disable Collections.

Dragging the font creates a copy of the original and a dot appears next to the font name to show that a duplicate exists. To remove the font from the Computer collection, select it and choose File > Remove Fonts.
