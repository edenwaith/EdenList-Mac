Previewing fonts in Font Book

You can use Font Book to preview the different typefaces and characters that are available in each of the fonts on your computer.

  1. Double-click Font Book in the Finder, or choose Manage Fonts from the Action pop-up menu in the lower-left corner of the Font panel.
  2. Select a collection in the Collection column and select the font family in the Font column that you want to preview.
  3. Click the disclosure triangle next to the font name to see which typefaces are available for that font.
  4. Select a typeface. A sample of the characters available in that typeface appear in the preview area to the right. If you don't see a preview area, choose Preview > Show Preview.
  5. To see how the selected typefaces appear at different font sizes, choose a font size from the Size pop-up menu, or drag the font size slider.
