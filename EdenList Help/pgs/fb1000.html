Making fonts available to all computer users

You can use Font Book to make fonts available to all the people who have accounts on your computer.

  1. Double-click Font Book in the Finder, or choose Manage Fonts from the Action pop-up menu in the lower-left corner of the Font panel.
  2. Select User in the Collection column, and then select the font in the Font column.
  3. Drag the font to Computer in the Collection column.

This creates a copy of the original and a dot appears next to the font name to show that a duplicate exists. To remove the font from the User collection, select it and choose File > Remove Fonts.
