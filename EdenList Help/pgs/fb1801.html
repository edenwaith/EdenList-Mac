Creating a new font collection

You can create your own font collections to help you manage and organize your font library.

  1. Open Font Book and choose File > New Collection.
  2. Type a name for the collection.
  3. Click the font library you want to use and drag fonts from the Font column to your newly created collection.

Adding a font to a collection doesn't move the font itself, but only places a pointer to the font in the collection. You can add the same font to as many collections as you want.
