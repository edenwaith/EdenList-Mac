Checking for damaged fonts

If you are having problems with installed fonts, you can check them to make sure they haven't become corrupted.

  1. Open Font Book and select the fonts you want to check.
  2. Choose File > Validate Fonts.
  3. In the Font Validation window, see if any warnings or errors are reported.
  4. Click the triangle next to a font to see which validation rules show errors.

If a font is damaged or corrupted and you want to remove it, click the checkbox next to the font. Make sure any fonts you want to keep are not checked, and then click the Remove Checked button.
