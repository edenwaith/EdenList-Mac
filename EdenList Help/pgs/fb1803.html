Automating Font Book tasks with AppleScript

Font Book includes AppleScript terms that let you automate tasks using scripts. You can use scripts to create a new collection, create an italic collection, delete an empty collection, or create a TextEdit document that shows all your installed fonts.

To see the Font Book terms and write scripts, you can use Script Editor, included with Mac OS X. To learn more about AppleScript, AppleScript terms, and Script Editor, see AppleScript Help.

To create your own scripts for Font Book:

Drag the Font Book icon onto the Script Editor icon, located in the /Applications/AppleScript folder.
